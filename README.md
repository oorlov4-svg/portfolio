🌐 Web Design Agency Portfolio

A modern, responsive portfolio website for a web design agency, showcasing projects and providing an intuitive way for potential clients to get in touch.

🚀 Features

🖼 Projects Gallery
Portfolio displayed as interactive project cards with categorized content (UI/UX, Landing Pages, E-commerce, etc.)

🔍 Project Filtering
Dynamic filtering of project cards without page reload for better UX

🪟 Project Modal with Slider
Each project card opens a modal window with a slideshow gallery implemented using jQuery

📩 Contact / Feedback Modal
Modal window with a contact form:

Email input field

Message / question textarea

Basic form validation

⚡ Interactive UI
Smooth animations, hover effects, and dynamic DOM manipulation

📱 Responsive Design
Fully responsive layout optimized for desktop, tablet, and mobile devices

🛠️ Tech Stack

HTML5

CSS3 (Flexbox / Grid)

JavaScript (ES6+)

jQuery

💡 Implementation Details

Component-based structure for reusable UI elements

Modal windows handled via JavaScript event delegation

jQuery used for DOM manipulation and slider functionality

Clean and maintainable code structure (BEM methodology)

Optimized assets and images

📬 Contact Flow

Users can open a modal form, enter their email and message, and send a request for feedback or project inquiry.

🎯 Project Goal

To create a visually appealing and user-friendly platform for presenting a web design agency’s work and improving client engagement through interactive UI elements.

🔧 Future Improvements

Form submission with backend integration (e.g., API / email service)

Accessibility improvements (ARIA attributes, keyboard navigation)

Migration from jQuery to modern frameworks (React / Vue)
